Unidad 1: Furniture

Muestra Virtual

¡Visita la muestra virtual preparada por el Taller de Productos!

La exposición recopila el trabajo de los estudiantes de las tres secciones del Taller de Productos en una muestra interactiva virtual. El encargo propuesto a los estudiantes se realiza en el marco de la Unidad 1: Furniture.

Para una mejor experiencia visitando la muestra virtual desde tu computador, puedes descargar el archivo ejecutable TDP-U1-ExhibicionVirtual.exe
